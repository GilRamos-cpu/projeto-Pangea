# Turma da Mata 
O Instituto PANGEIA “Panorama Geral de Integração Ambiental” foi criado no ano de 2005 para desenvolver projetos de plantio e criação de mudas para reflorestamento com o objetivo de capacitar crianças e adolescentes em situação de vulnerabilidade e torná-los social e financeiramente independentes.
